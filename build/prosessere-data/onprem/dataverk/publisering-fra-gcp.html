Publish data packages from GCP projects

When setting up on-demand GCP projects some dataverk configuration is automatically stored as secrets in Google Secret Manager in the created project. These configuration values needs to be imported and set as environment variables in order to use dataverk to publish data packages from GCP.

To simplify this, we have developed a python library dataverk-gsm which can be used as follows:

from dataverk_gsm import api as gsm_api
gsm_api.set_secrets_as_envs()
info

In order to publish data packages to the internal data catalog from GCP projects it is required that installed version of dataverk is >=0.4.4
